# GCU Lahore Website Clone

A static website inspired by the GCU Lahore website, built using HTML. This project focuses on webpage structure, navigation design, content organization, and form layout development.

## Features

* University-style homepage layout
* Header with logo and branding
* Navigation menu
* Hero banner section
* Contact information section
* Contact form layout
* Location information section
* Footer section
* Responsive content organization

## Technologies Used

* HTML5

## Project Purpose

This project was developed to strengthen my understanding of HTML fundamentals, webpage structuring, content layout design, and website development concepts.

## Skills Practiced

* HTML page structure
* Table-based layouts
* Navigation design
* Form creation
* Image integration
* Content organization
